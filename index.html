Notes on the web & basic HTML

The basics of the web and HTML - Important stuff to remember

A brief introduction to the web

The web is mostly made up of HTML (HyperText Markup Language) documents with Hyperlinks (links to other webpages makes the web web like). It was invented in the 1990's, and there are over 30 million pages on the web (though which indexing system this figure is using I don't know!). Files on the web include:

  • Plain Text
  • HTML (the main language of the web)
  • Images
  • Videos
  • Music

    • So what is the web?

    When you visit a webpage such as udacity.com, your computer sends a request to a server using HTTP (the main protocol of the web), the server finds the HTML document you are requesting and sends it back to you, where your browser displays the content of the file(s).

    HTML - HyperText Markup Language

    HTML is made up of:

  • Text content - which is what you see
  • Markup - which is what the content looks like
  • References to other documents - such as images and videos
  • Links to other pages- also known as Hyperlinks

    • HTML Markup is made up of tags, these are used to define the structure of the content displayed on an HTML page, such as text, images, and videos. Most (but not all) tags have an opening tag and a closing tag, and in between them is your content. This structure is referred to as an Element. Two such elements that define the structure of text displayed on a HTML page are:

  • Bold - <b>Text goes here</b>
  • Emphasis - <em>Text goes here</em>

    • TIP! It's often a good idea to make your closing tags as soon as you make your opening tag so that you don't cause unexpected problems. Computers are stupid and they will only do what you tell them to do. They can not intuit what you mean. If you forget a closing tag then the opening tag will be attributed to everything after it. The computer does not know where you meant to put the closing tag, computers are not psychic!

    HTML Attributes:

    • Whitespace:

    Text on multiple lines within the HTML appear as one line in the browser, the excess whitespace is stripped. If you want text to appear on multiple lines then you can use the following elements:

  • Break (void) - <br>

    So if <br> and <p> do the same thing, why do we have both?
  • <br> is an inline element which is good for ending a line.
  • <p> is a block element, it makes an invisible box around text, which can have height and width. This is good when you want the block of text to have a bit of padding around it.

    • Container Elements:

    Are elements that do nothing but contain stuff inside themselves, these are used to control how text is laid out. The following are examples of container elements:

  • <span> is an inline element, it is used to group inline elements together, it also allows styling to be applied to text contained in the span.

  • <div> is a block element, it is used for layout, styling and positioning of text on a page (most often using CSS).
